# Support Policy

`batchor` support currently runs through GitHub Issues.

Supported scope:

- bug reports against the latest released `0.x` minor
- documentation corrections and clarifications
- packaging and install issues for supported Python versions

Not supported:

- older `0.x` minors after a newer minor is released
- unreleased branches or local forks
- requests for commercial licensing or commercial support through this repository

When opening an issue, include:

- package version
- Python version
- platform
- storage backend details if relevant
- minimal repro or command sequence
